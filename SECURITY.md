# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it privately. Do not open a public issue.

Please report vulnerabilities to the maintainer directly or via GitHub's private vulnerability reporting feature if enabled.

We will acknowledge receipt of your report within 48 hours and provide a timeline for a fix.
